Esri Labs

The Esri Labs website was not simply a website designed for a desktop browser interface but was based on responsive or adaptive environment. Using CSS3 media queries and fluid grid concept the web content adjusted itself to the resolution of the browser screen. All the information reconfigured, layouts relaid, the typography changed suitable and selective content loaded to address optimization issues. It was one piece of code which worked across all devices replacing the need for native implementation.
